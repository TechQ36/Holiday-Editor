# Holiday-Editor

让博客作者将精力集中在文章内容，早用早放假，妈妈再也不用担心我写文章不想排版了！

## 简介

基于 Electron 的 Markdown 转译工具，支持多种代码主题/文章主题，支持自定义代码样式/文章样式，逐步更新，欢迎 star！

## 使用指南

### 安装依赖

```bash
npm install
cd src/render
npm install
```

### 运行

```bash
npm run start:react
npm run start:electron
```
